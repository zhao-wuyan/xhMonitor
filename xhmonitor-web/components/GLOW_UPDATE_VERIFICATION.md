## ⚠️ 已知问题

### 1. 构建产物未更新

**问题**: `dist/assets/index-*.js` 仍包含旧值 0.25

**影响**: 如果使用构建后的文件，光晕效果仍为旧值

**解决方案**: 
```bash
cd xhmonitor-web
npm run build
```

### 2. 浏览器缓存

**问题**: 浏览器可能缓存旧的 CSS 文件

**解决方案**: 
- 硬刷新: Ctrl + F5 (Windows) 或 Cmd + Shift + R (Mac)
- 清除缓存
- 使用无痕模式
